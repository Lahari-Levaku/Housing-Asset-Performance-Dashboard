# Housing-Asset-Performance-Dashboard
Power BI dashboard for monitoring housing Asset Condition, Compliance Status and Risk Level

📊 Project Overview
This project showcases an interactive dashboard built in Power BI to monitor housing asset performance, focusing on property condition, compliance, and risk levels.

🎯 Key Objectives: 
* Track property condition ratings
* Identify high-risk assets
* Monitor compliance status
* Highlight overdue inspections
* Analyze inspection trends over time

📈 Dashboard Features: 
* Dynamic filters by location, risk level, and property type
* KPI metrics for quick insights
* Condition distribution analysis
* Monthly inspection trends
* High-risk asset tracking table

📊 Key Insights:
* 200 total properties monitored
* 40 high-risk assets identified
* 67% compliance rate
* 122 overdue inspections
* Average condition score: 2.83

🛠️ Tools & Technologies: 
* Power BI
* Data cleaning and transformation
* Data visualization techniques
  
💡 Business Impact:
This dashboard helps stakeholders quickly identify non-compliant and high-risk properties, enabling better prioritization of maintenance and inspections.
